# نموذج إضافة كتاب جديد

## تعليمات

هذا ملف نموذجي يوضح الهيكل الكامل لإضافة كتاب جديد إلى المكتبة.

### البيانات الوصفية (Front Matter)

```yaml
---
layout: book              # استخدم دائماً "book"
book_id: book-unique-id   # معرّف فريد للكتاب (بدون مسافات)
title: عنوان الكتاب      # العنوان بالعربية
author: اسم المؤلف       # اسم صاحب الكتاب
section: hadith          # القسم: quran, hadith, tafseer, fiqh, history
description: وصف قصير   # وصف يُظهر في قائمة الكتب
pages: 150              # عدد الصفحات التقريبي
content_type: single    # single = ملف واحد, multi-chapter = فصول
---
```

## هيكل الملفات

### للكتب البسيطة (ملف واحد):
```
books/book-id/
└── index.md
```

### للكتب الكبيرة (عدة فصول):
```
books/book-id/
├── index.md
└── chapters/
    ├── chapter-1.txt
    ├── chapter-2.txt
    └── ...
```

## أقسام متاحة

| القسم | الرمز | مثال |
|-------|-------|------|
| القرآن الكريم | `quran` | الزيارة الجامعة |
| الأحاديث | `hadith` | نهج البلاغة |
| التفسير | `tafseer` | تفسير العياشي |
| الفقه | `fiqh` | أصول الفقه |
| التاريخ | `history` | سيرة الإمام الحسين |

## تحديث قائمة الكتب

بعد إنشاء الملف، أضف إدخال في `_data/books.yml`:

```yaml
- id: book-id
  title: عنوان الكتاب
  author: اسم المؤلف
  section: hadith
  description: وصف قصير
  pages: 150
  content_type: single
  order: 11  # الترتيب في القائمة
```

## الاختبار المحلي

```bash
cd /path/to/mktba
bundle exec jekyll serve
```

ثم افتح http://localhost:4000/mktba
